# Stopwatch Web App

A simple stopwatch web application built using HTML, CSS, and JavaScript.  
The application allows users to start, pause, and reset the timer.

## Features
- Start the stopwatch
- Pause the stopwatch
- Reset the timer
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure

Stopwatch-Web-App

│

├── index.html

├── style.css

└── app.js

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open *index.html* in a browser
